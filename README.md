# routine-manager-app

A web application for managing and tracking maintenance routines of machinery in manufacturing plant.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Testing:** Vitest, React Testing Library

## Features

- Real-time data updates
- Complete CRUD operations for main resources
- Responsive UI/UX design

## Getting Started

Follow these instructions to start using the app in your plant.

### Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/)
- Access to a remote database (e.g., MongoDB Atlas) or a local database instance
