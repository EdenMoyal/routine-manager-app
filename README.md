# routine-manager-app

A web application for managing and tracking maintenance routines of machinery in manufacturing plant.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Testing:** Vitest, React Testing Library

## Features

- Real-time data updates
- Complete CRUD operations for main resources
- Responsive UI/UX design
